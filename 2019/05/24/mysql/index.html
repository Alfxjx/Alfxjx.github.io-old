在WAMP环境操作MySQL数据库

2019-05-24

文章目录

  1. 准备工作
  2. 操作数据库
  3. CRUD

今天学习了一下怎么使用 wamp 链接 mysql 数据库。

准备工作

首先是打开 wamp server, 点击状态栏的图标，选择MySQL->MySQL console;

输入use mysql

浏览器中打开localhost

在界面中选择phpmyadmin

reference

操作数据库

选择新建， 即可新建一个数据库。排序规则选择utf8_general_ci，貌似这个比较常用。

接着就是新建数据表，配置字段数。

可以看到一个配置界面，愉快的配置吧。

PS：设置主键的时候，需要将类型设置为INT,并且勾选A_I(自增)。

CRUD

CRUD 就是增删改查的意思。

  • 增加 选择插入的 tab 即可 PPS: 插入的时候，主键不需要填数字嗷。
  • 查询 选择搜索里面可以根据条件检索
  • 修改和删除 在查询之后就可完成
