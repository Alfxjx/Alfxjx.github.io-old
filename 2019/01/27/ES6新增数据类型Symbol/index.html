ES6新增数据类型Symbol

Jan 27 2019

es6中新增一种原始数据类型Symbol,最大的特点是唯一性，Symbol值通过Symbol函数生成, 在es5中对象的属性都是字符串，我们使用他人定义的对象，然后去新增自己的属性，这样容易起冲突覆盖原有的属性, Symbol也可以看成为一个字符串，不过这个字符能保证是独一无二的

基本示例:

// Object
const obj = {
    name:'i am object'
}
// Symbol
// 这里的参数没有特殊意义，可以看成为Symbol加了一个标记
const name = Symbol('name')
obj[name] = 'i am symbol'

Symbol用法

Symbol 目前有多种写法

const title = Symbol()

const obj = {}
obj[title] = 'Symbol'

const obj = {
    [title]: 'Symbol'
}

Object.defineProperty(obj,title,{value:'Symbol'})

obj[title]// 输出的结果都为Symbol

这里注意一下，Symbol做为属性名时，不能用点去读取，obj.title // undefined

Symbol作为属性名，只有通过Object.getOwnPropertySymbols方法返回

const attrs  =Object.getOwnPropertySymbols(obj)
// [Symbol[title]]

Symbol.for()

如果我们想要重复Symbol可以使用Symbol.for， Smybol.for()及Smybol()的区别在于Symbol.for()会先去查找全局下有没有声明过，如果有就返回该值，没有就新建一个，Symbol()每一次调用都会新建一个

代码理解:

const title = Symbol.for('symbol')
....Symbol.for('symbol')//调用多次都是使用的第一次声明的值
Symbol.for('JS每日一题')  === Symbol.for('JS每日一题')  // true
const title = Symbol('symbol')
Symbol('JS每日一题')  === Symbol('JS每日一题')  // false

总结

Symbol的特点

  • 独一无二
  • 不能隐式转换
  • 不能与其它数据类型做运算
  • 不能使用点运算符进行操作
